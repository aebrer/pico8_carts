Controls & Technical Notes

Arrow Keys: Navigate the map directly (more control than Beginner)

Z + Arrow Keys: Hop between metalayer seeds (entropy network prevents adjacency)

X: Take screenshot

Z + X: Record gif

Enter: Open menu with control list
