Description

Entropy locking meets wave function collapse algorithm, plus ideocartography. A screensaver exploration with bombastic colors—gods and temples emerging from controlled chaos.

Author's Notes

Originally titled "BEHOLDER" and "GODS AND TEMPLES," this piece takes the ideocart concept into screensaver territory. The more bombastic color palette shifts the pareidolia pattern—faces and entities give way to architectural forms, divine structures, temples rising from the noise.

Experience visions in a square for highest resolution. Scales to any display—once fullscreen, refresh to adjust to new resolution.

Technical Notes

Built with p5.js. Combines three core techniques:

  • Entropy locking - Probabilistic RNG reseeding creates controlled chaos
  • Wave function collapse - Algorithmic pattern generation with constraint propagation
  • Ideocartography - Designed to trigger pareidolia—you will see things

Designed for indefinite runtime as an ambient screensaver. The pattern never quite repeats.
