Author's Notes

This piece is part of the marblez project by @jmh_wwyg - "digital fantazy marblez" created in Game Maker Studio with retro PSX-esque graphics. You can move the marble around and check out the properties and objects stored inside. A lot of them have secret properties, and this one is no exception. Try viewing it at specific times.

The concept (for me) was that the marble is almost like a standard weight for a quantified unit of Dead God Pulse energy/frequency. But as you can see, the marble is cracked, and maybe containment isn't as reliable as the Foundation imagined.

This is a secret member of the vestiges series - we forgot to tag it on release, so it's never been officially categorized until now.

I had to create the looping gif(s), ensure perfect loops, make multiple versions (one relates to the time-based secret), and coordinate with @jmh_wwyg's technical requirements for the Game Maker integration. They did incredible work with the planar mapping to give my flat 2D gifs a 3D presence in the final piece.

The source code for my contribution is lost - these were transient gifs like in Local Ontology Secured Transistor, created specifically for this collaboration and not preserved separately.
